# Rock-Paper-Scissors Game

A simple Rock-Paper-Scissors game built with HTML, CSS, and JavaScript.

## Live Demo

You can play the game live at: https://kashukartik.github.io/Rock-Paper-Scissors/

## Game Description

This game allows you to play Rock-Paper-Scissors against the computer. The game randomly selects the computer's choice and determines the winner based on the game's rules.

## Features

* Play against the computer
* Randomly generated computer choice
* Displays the game result (win, lose, or tie)

## Technologies Used

* HTML
* CSS
* JavaScript

## How to Play

1. Open the game in your web browser
2. Click on one of the buttons (Rock, Paper, or Scissors) to make your choice
3. The computer's choice will be randomly generated and displayed
4. The game result will be displayed on the screen

## Code Structure

The code is organized into three main files:

* `index.html`: The main HTML file that contains the game structure
* `style.css`: The CSS file that styles the game
* `script.js`: The JavaScript file that contains the game logic
